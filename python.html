🐍 Python基础课程

从零开始学习 Python 编程

课程大纲

01

Python 环境搭建

安装 Python 解释器，配置开发环境（VS Code / PyCharm），编写第一个 Hello World 程序。

  • Python 下载与安装
  • IDE 配置
  • 运行第一个程序
02

变量与数据类型

学习 Python 中的基本数据类型：整数、浮点数、字符串、布尔值，以及变量的定义与使用。

  • 整型 int
  • 浮点型 float
  • 字符串 str
  • 布尔型 bool
03

运算符与表达式

掌握算术运算符、比较运算符、逻辑运算符和赋值运算符的使用。

  • 算术运算符
  • 比较运算符
  • 逻辑运算符
  • 成员运算符
04

条件语句

学习 if / elif / else 条件判断语句，实现程序的分支逻辑控制。

  • if 语句
  • if-else 语句
  • if-elif-else 语句
  • 嵌套条件
05

循环语句

掌握 for 循环和 while 循环，学习 break、continue 控制流程。

  • for 循环
  • while 循环
  • break 与 continue
  • 循环嵌套
06

列表与元组

学习列表（List）和元组（Tuple）的创建、访问、切片和常用操作方法。

  • 列表创建与访问
  • 列表切片
  • 列表方法
  • 元组特性
07

字典与集合

掌握字典（Dict）和集合（Set）的使用，理解键值对和去重的概念。

  • 字典操作
  • 集合操作
  • 推导式
08

函数

学习函数的定义与调用、参数传递、返回值以及 lambda 表达式。

  • def 定义函数
  • 参数类型
  • 返回值
  • lambda 表达式
09

文件操作

学习文件的读取、写入、追加操作，以及 with 语句的使用。

  • open() 函数
  • 读取文件
  • 写入文件
  • CSV 文件处理
10

模块与包

了解 Python 模块化编程，学习 import 机制和常用标准库。

  • import 导入
  • 常用标准库
  • 第三方库安装
  • pip 工具使用
